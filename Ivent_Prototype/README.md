# City Events (прототип) — Next.js + Prisma + Postgres

Це навчальний прототип "Сайт івентів міста" для практики: каталог подій, календар-місяць, сторінка події, відгуки, купівля квитків (створення замовлення), ролі (USER/ORGANIZER/ADMIN) та панель організатора (/admin).

## Швидкий старт (локально)

### 1) Вимоги
- Node.js 18.17+ (краще 20+)
- Docker Desktop (для Postgres) або встановлений Postgres локально

### 2) Запуск БД (Docker)
У корені проєкту:
```bash
docker compose up -d
```

### 3) Налаштування .env
Скопіюй приклад:
```bash
cp .env.example .env
```

### 4) Встановити залежності
```bash
npm install
```

### 5) Міграції та seed (тестові дані)
```bash
npx prisma migrate dev -n init
npx prisma db seed
```

### 6) Запуск застосунку
```bash
npm run dev
```

Відкрити: http://localhost:3000

## Сторінки
- / — каталог подій
- /calendar — календар-місяць
- /events/[id] — сторінка події (квитки, відгуки)
- /dashboard — кабінет (мої замовлення)
- /admin — панель організатора/адміністратора (створення подій і квитків)
- /login — вхід

## Примітка про "купівлю квитків"
У прототипі оплата не інтегрована. Натискання "Оформити замовлення" створює запис Order у БД та зменшує доступну кількість квитків.
